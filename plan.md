# Upper

## 프로젝트 개요

- **목적**: 미용(피부과, 성형외과, 한방, 치과) 분야 의사 검색 및 상세 정보 제공 서비스
- **이름**: Upper

## 기술 스택

- **Frontend**: Next.js (App Router 권장), React, TypeScript, Tailwind CSS, Lucide React (아이콘)
- **Backend/Database**: Supabase (Auth, PostgreSQL)
- **File**: Supabase Storage

## 핵심 기능

- 의사 목록 필터링 (세부 시술/치료 분야, 지역 등)
- 병원 정보 (이름, 주소, 로고)
- 의사 상세 페이지: 이름, 사진, 소속 병원 (이름, 주소, 홈페이지), 추구미 목록, 진료 철학, 임상 경력 목록, 전문의 취득 후 임상 경력 목록, 전문 분야 목록, 구사 혹은 통역 언어 목록
- 검색 기능 (지역, 이름, 진료 분야, 시술 카테고리, 시술 등으로 검색)
- 관리자 페이지: 의사/병원 데이터 추가/수정/삭제, 의사 추가 하면서 병원 지정하거나 새로운 병원 정보 추가 기능
- 인증: 사용자 관리, 회원가입(signup), 구글 로그인(회원 가입한 경우만), "어드민"인 경우만 어드민 메뉴 진입 및 정보 편집 가능

# UI
- 당근마켓(www.daangn.com) 메인 페이지와 유사하게 구현
- 세련된 UI 구현
- "믿을 수 있는 의사를 찾아보세요" 밑에 검색창 넣기
- 그 아래에는 "눈 성형", "지방 성형", "보톡스" 등 시술 카테고리 키워드를 나열하기
- 그 아래에는 미용 분야 주요 지역을 키워드로 나열하기
- 검색하거나, 전문 분야, 지역 키워드를 선택하면 의사 검색 페이지로 이동
- 의사 검색 페이지에 메인 페이지와 동일한 검색창 상단에 배치하고 검색된 의사는 수평으로 긴 박스 형태로 보여주기
- 의사 박스 클릭 시 의사 상세 정보 페이지로 이동
- 의사 검색 페이지의 각종 필터는 좌측 패널에 드롭다운이 아니라 나열식으로 표시
- 병원 주소는 검색을 통해 입력해야하고 "서울시 강남구 삼성로 14, 406동 1605호 (개포동, 개포자이 프레지던스)" 형식으로 표현
